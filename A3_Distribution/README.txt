# A3 Document Automation - Distribution Package

## Installation Instructions:

1. **Simple Installation:**
   - Double-click `install_a3.bat`
   - This will install the application and create a desktop shortcut

2. **Manual Installation:**
   - Copy `A3_Automation.exe` to your desired location
   - Run the executable directly

## Features:
- Automatic document processing with GPT-4o OCR
- Custom PDF template generation with A3_Custom_Template.pdf
- Interactive field positioning
- Auto-update functionality from GitHub

## System Requirements:
- Windows 10 or later
- Internet connection (for OCR and updates)
- OpenAI API key

## First Time Setup:
1. Install using install_a3.bat or run A3_Automation.exe
2. The launcher will open - click "Launch A3 Automation"
3. Configure your OpenAI API key when prompted
4. Process your first A3 document!

## How It Works:
1. Launch the application from desktop shortcut
2. Select your handwritten A3 document (PDF/image)
3. The system extracts text using GPT-4o OCR
4. Text is automatically populated into A3_Custom_Template.pdf
5. Final output saved to processed_documents folder

## Auto-Updates:
- App automatically checks for updates on startup
- Downloads latest version from GitHub
- One-click update installation
